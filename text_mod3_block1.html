Рюкзачные криптоситсемы

Задача о рюкзаке

В 1978 году Ральф К. Меркл и Мартин Э. Хеллман, оба инженеры-электрики из Стэнфордского университета, разработали криптосистему с открытым ключом, основанную на задаче о рюкзаке, знаменитой задаче в комбинаторике.

Ее можно сформулировать следующим образом: учитывая рюкзак объемом S и n предметов различных объемов a1, a2, ..., an , какой из предметов может заполнить рюкзак?

Другими словами, учитывая положительные целые числа a1, a2, ..., an, называемые весами, и положительное целое число S, решим (1)

S = a1·x1 + a2·x2 + ... + an·xn (1)

где xi = 0 или 1, а S является точечным произведением векторов(a1, a2, ..., an) и (x1, x2, ..., xn)

Проблема с рюкзаком может

  • не иметь решений
  • иметь одно единственное решение
  • иметь несколько решений

Например, проблема с рюкзаком

3x1 + 5x2 + 9x3 + 19x4 + 37x5 = 45

имеет одно решение (1, 1, 0, 0, 1), так как

3 + 5 + 0 + 0 + 37 = 45

С другой стороны, проблема с рюкзаком

3x1 + 5x2 + 8x3 + 13x4 + 31x5 = 34

имеет два решения: (0, 0, 0, 1, 1) и (0, 1, 1, 0, 1), так как

0 + 0 + 0 + 13 + 21 = 34 = 0 + 5 + 8 + 0 + 21

Но проблема

5x1 + 14x2 + 15x3 + 27x4 + 11x5 = 23

не имеет решений.

Задача (1) может быть решена довольно легко, если веса обладают особыми свойствами. Например, если

ai = 2i-1

, то

S = x1 + 2x2 + 22x3 +···+ 2n-1xn

имеет решение (x1, x2, ..., xn) если (xn, xn-1, ..., x1) = S

Ее также легко решить, если

∑i=1j-1ai < an, 2 ≤ j ≤ n

Последовательность с этим свойство называется сверхвозрастающим. Например, рассмотрим последовательность 3, 5, 9, 19, 37. Так как 3 < 5, 3 + 5 < 9, 3 + 5 + 9 < 19, и 3 + 5 + 9 + 19 < 37 , то последовательность сверхувеличивается.

Следующий пример показывает, как решить задачу о рюкзаке с помощью увеличения веса.

Решим задачу о рюкзаке

3x1 + 5x2 + 9x3 + 19x4 + 37x5 = 45.

Так как веса сверхувеличиваются и поскольку 3 + 5 + 9 + 19 < 37 < 45, то

x5 = 1

Тогда

3x1 + 5x2 + 9x3 + 19x4 = 8

Так как 19 > 8 и 9 > 8, то

x4 = x3 = 0

что дает

3x1 + 5x2 = 8

Наконец,

x1 = x2 = 1

Таким образом, решением данной проблемы является (1, 1, 0, 0, 1)
