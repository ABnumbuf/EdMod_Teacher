Задача дискретного логарифмирования

Метод согласования

Метод согласования, предложенный в 1962 году Александром Осиповичем Гельфондом (1906 - 1968), является одним из первых методов решения задачи дискретного логарифмирования. Этот метод реализует алгоритм решения сравнения

ax ≡ b (mod n)

Позже этот алгоритм был независимо описан Дэниелом Шенксом (1917-1996) и получил название метод больших и малых шагов.

Метод основан на следующем хорошо известном теоретико-числовом факте:

Пусть p ∈ P\{2} и h = [√ p ] тогда для любого целого числа x, 0 ≤ x ≤ p, найдутся такие целые числа l, t, x = hl - t, 0 < l ≤ h, 0 ≤ t < h

(Для его доказательства достаточно рассмотреть l = [ x⁄h ] и t = hl - x

В этих условиях для нахождения числа x из ЗДЛ воспользуемся полученным разложением:

b ≡ ax ≡ (ah)la-t (mod p)

или

bat ≡ (ah)l (mod p)

Это означает, что искомые значения l, t могут быть найдены перебором величин l, t и l, t в указанных границах, а после этого будет определена и величина x.

Алгоритм метода согласования заключается в следующем:

  1. Для всех возможных значений t = 0, ... , h - 1 вычислим вычеты bat
  2. Для всех возможных значений l = 1, 2, ... , h вычислим вычеты (ah)l (mod p)
  3. Начинаем сравнивать значения из первой последовательности со значениями из второй последовательности. Как только будет найдено совпадение элемента из второго множества с некоторым элементом первого множества, мы определим неизвестные l, t и вычислим

    4. x = hl - t
