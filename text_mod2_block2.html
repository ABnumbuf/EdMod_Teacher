Задача дискретного логарифмирования

Метод Сильвестра-Полига-Хеллмана

Метод Сильвестра-Полига-Хеллмана нахождения дискретного логарифма x из сравнения (1) был предложен в 1965 году Василием Ильичем Нечаевым (1920 - 1999).

ax ≡ b (mod n) (1)

Этот метод позволяет в случае составного показателя Pp(a) свести задачу вычисления дискретного логарифма к нескольким таким подзадачам. Именно, при Pp(a) = p1a_1 p2a_2... pka_k решение задачи поиска целого неотрицательного числа x, удовлетворяющего сравнению (2) можно свести к решению системы сравнений (3).

ax ≡ b (mod p), 0 ≤ x < Pp(a) (2)

{ x ≡ x1 (mod p1a_1) (3)
...
x ≡ xk (mod pka_k)

Искомое x следует вычислить, пользуясь китайской теоремой об остатках.

Рассмотрим алгоритм Полига-Хеллмана. На входе необходимо простое число p и вычеты a, b, удовлетворяющие сравнению (1). А также выполняться равенство (4). На выходе получаем дискретный логарифм (5).

ordpa = qn, n ∈ N, n > 1 (4)

x = logab (mod qn) (5)

Сначала определяем (6). Далее для всех i от 1 до n-1 выполняем (7).

an-1 = a, bn-1 = b (6)

an-i-1 ≡ an-iq (mod p), bn-i-1 ≡ bn-iq (mod p) (7)

Затем для всех i от 0 до n-1 проверить: если i > 0, то найти (8), иначе γ = 1.

Определяем (9) и, с помощью, например, метода согласования, находим дискретный логарифм (11), удовлетворяющий сравнению (10).

γ ≡ aix_0 ai-1x_1 ... a1x_i-1 (mod p) (8)

βi ≡ bi γ-1 (mod p) (9)

a0x_i ≡ βi (mod p) (10)

x = x0 + x1q + ... + xn-1qn-1 (11)

Задача дискретного логарифмирования решена.
